AddRss - это опенсорс решение для чтения RSS и ATOM новостных лент.
После определения всех необходимых подписок, Телеграм бот будет рассылать уведомления при появлении новых записей. 

Используйте /list для отображения списка текущих подписок.
Или /add [адрес] для того чтобы добавить ленту в подписки. Адрес должен начинаться с http:// или https://.
И /remove [имя] для удаления ленты из подписок.

Это сообщение можно повторить при помощи команды /start.
Исходный код доступен по ссылке https://github.com/VladikAN/addrss-telegram.
